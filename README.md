# APP-001 Coach Espiritual

App web estática desplegada en GitHub Pages. Muestra podcasts y meditaciones diarias del Coach Espiritual, organizados por rotación semanal de ejes temáticos.

## Rotación semanal

| Día | Eje temático |
|-----|-------------|
| Lunes | Humildad |
| Martes | Servicio |
| Miércoles | Fe y Oración |
| Jueves | Presencia |
| Viernes | Compasión |
| Sábado | Gratitud |
| Domingo | Trascendencia |

## Configuración

Al abrir por primera vez, introducir la API key de Google Drive (lectura pública) en el panel de configuración.

## Tecnología

- HTML5 + CSS3 + JavaScript vanilla
- Google Drive API v3 (lectura pública)
- GitHub Pages (hosting gratuito)
- Sin backend, sin dependencias
